Sección 1

¿Qué es Tinkercad?

🌐
Es un simulador online

Tinkercad es una plataforma gratuita de Autodesk que permite diseñar y simular circuitos electrónicos desde el navegador.

🔌
Arduino virtual

Podés usar una placa Arduino, componentes, cables y protoboard de forma digital.

🧱
Bloques de programación

Permite empezar sin escribir código desde cero, usando bloques visuales.

🛡️
Seguro para aprender

Si algo falla, se corrige en la simulación sin dañar componentes reales.

Sección 6

Errores frecuentes

🔃
LED invertido

El LED no enciende.

Solución: revisar ánodo y cátodo.
🚫
Falta la resistencia

Puede aparecer una advertencia.

Solución: usar una resistencia de 220 Ω.
📍
Pin incorrecto

El cable y el bloque no coinciden.

Solución: verificar que ambos usen el pin 8.
⚫
Falta GND

El circuito no se cierra.

Solución: conectar el cátodo a GND.
